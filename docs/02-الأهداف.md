# تسليم: صفحة الأهداف

## الملفات المعدّلة/المضافة
- `src/core/useCore.tsx` — توسعة النواة بأنواع ودوال الأهداف.
- `src/pages/Goals.tsx` — الصفحة الجديدة (إضافة).
- `src/styles/global.css` — شريط تقدّم عام + أصناف الأهداف (كل لون من `:root`).
- `src/App.tsx` — مسار `/goals`.

## ما نُفّذ
- إضافة/تعديل (بالضغط المباشر)/حذف أهداف.
- لكل هدف **خطوات** (مهام فرعية): إضافة/تعديل/إنجاز/حذف.
- **شريط تقدّم** يعرض نسبة الخطوات المنجزة.
- **اكتمال تلقائي** للهدف عند إنجاز كل خطواته → شارة «✅ مكتمل» + نبضة + احتفال.

## النقاط (XP) عبر النواة الموحّدة
- إنجاز خطوة (مهمة فرعية) = **+3** (عند الانتقال إلى "تم").
- اكتمال هدف كامل = **+25 + Confetti**.
- الهدف **يُقفل عند الاكتمال** (`completed`) لمنع تكرار احتساب +25.

## قائمة الفحص
- XSS ✅ — نصوص `{متغير}` فقط، لا `innerHTML`.
- ألوان `var(--)` ✅ — لا لون ثابت.

- لا `prompt`/`confirm` ✅ — حذف عبر `ConfirmDialog`.
- `crypto.randomUUID()` ✅ — لكل هدف وخطوة.
- تنظيف المدخلات ✅ — `clean()` ≤200 حرف.

- تجاوب + احتفال 100% ✅.

# تسليم: صفحة الروتين الصباحي/المسائي

## الملفات المعدّلة/المضافة
- `src/core/useCore.tsx` — توسعة النواة بأنواع ودوال الروتين.
- `src/pages/Routine.tsx` — الصفحة الجديدة (إضافة).
- `src/styles/global.css` — أصناف الروتين (كل لون من `:root`).
- `src/App.tsx` — مسار `/routine` + زر وصول من الرئيسية.

## ما نُفّذ (وفق المواصفة)
- **مهام دائمة لا تُحذف يومياً** — فقط حالة "تم اليوم" تُصفَّر تلقائياً (مقارنة `doneDate` بتاريخ اليوم).
- **تعديل بالضغط المباشر** على نص المهمة/الفرعية (حقل إدخال أنيق، حفظ بـ Enter أو فقدان التركيز).
- **أولوية بنقطة ملوّنة دورية**: 🟡 متوسطة → 🔴 عالية → 🟢 منخفضة (ألوان `--warning/--danger/--success`).
- **ترتيب حسب الوقت**: قبل 16:00 يُفتح الصباحي، بعدها المسائي.
- **إضافة بلا حد أقصى** لكل قسم.
- **مهام فرعية** تحت كل مهمة (إضافة/تعديل/إنجاز/حذف) قابلة للطيّ/الفتح.

## النقاط (XP) عبر النواة الموحّدة
- إنجاز مهمة روتين = **+5** (مرة واحدة عند الانتقال إلى "تم"، لا تتكرر).
- إنجاز مهمة فرعية = **+3** (مرة واحدة).
- عند إكمال كل مهام القسم 100% → **احتفال (Confetti) + نبضة** + `markStreakToday()`.
- إكمال الفرعيات لا يُكمل الأم تلقائياً (المستخدم يضغط "تم" بنفسه) — حسب الموافقة.

## قائمة الفحص
- XSS ✅ — كل النصوص عبر `{متغير}`، لا `innerHTML`/`dangerouslySetInnerHTML`.
- ألوان `var(--)` ✅ — لا لون ثابت في الصفحة أو الأصناف.
- لا تكرار ✅ — `updateSection` يوحّد الوصول للقسم، أصناف CSS مشتركة.
- لا `prompt`/`confirm` ✅ — الحذف عبر `ConfirmDialog`، الإضافة بحقول.
- `crypto.randomUUID()` ✅ — لكل مهمة وفرعية.
- تنظيف المدخلات ✅ — `clean()` (trim + توحيد مسافات + ≤200 حرف).
- `useCore` مركزي ✅ — لا حالة نقاط مستقلة.
- تجاوب جوال + احتفال 100% ✅.
- البناء ✅ — `npm run build` ينجح بـ TypeScript strict.

# المراجعة النهائية الشاملة + خطوات الرفع على Manus

## ✅ 1. قائمة المراجعة النهائية

### الأمان
- **XSS**: لا يوجد أي `innerHTML` أو `dangerouslySetInnerHTML` في الكود — كل النصوص عبر `{متغير}`.
- **المعرّفات**: كل المعرّفات بـ `crypto.randomUUID()`.
- **تنظيف المدخلات**: `clean()` (trim + توحيد المسافات + حد 200 حرف) لكل نص، و`clampNum()` لكل رقم (منع السالب والقيم الشاذة).
- **رفع الصور**: تحقق من نوع الملف (صورة) + حد حجم 1.5MB.
- **التبعيات**: `npm audit` → **0 ثغرات**.

### الأداء
- بناء Vite للإنتاج ينجح (`npm run build`) بـ TypeScript strict mode.
- حجم الحزمة ~85KB gzip — خفيف.
- مؤقت الراحة بـ timestamp (لا `setInterval` يتعطل بالخلفية).

### صفر تكرار
- كل CSS المشترك في `global.css` فقط، كل لون متغير من `:root`.
- دوال محوِّلة موحّدة (`updateSection`, `updateGoals`, `updateCustom`) ومكوّنات مشتركة.
- نظام XP موحّد عبر `useCore()` — لا حالة/عداد مستقل في أي صفحة.

### التجاوب والتجربة
- تصميم جوال أولاً (max-width 480px موسّط)، وضع ليلي + 5 ثيمات.
- احتفال بصري (Confetti) ونبضة عند إكمال أي قسم 100%.
- كل حذف عبر `ConfirmDialog` (لا `prompt`/`confirm`).

## 🚀 3. خطوات الرفع على Manus

### أ. التجهيز محلياً
1. تأكد أن البناء ينجح: `npm install` ثم `npm run build` (ينتج مجلد `dist/`).
2. الكود مرفوع بالفعل على GitHub: `mufkirat-alnajah`.

### ب. النشر عبر Manus
1. ادخل منصة Manus → أنشئ مشروعاً جديداً من نوع **Static Site / Vite + React**.
2. اربط مستودع GitHub `mufkirat-alnajah` (أو ارفع المجلد يدوياً).
3. إعدادات البناء:
   - **Build command:** `npm run build`
   - **Output directory:** `dist`
   - **Install command:** `npm install`
   - **Node version:** 20 أو أحدث.
4. اضغط Deploy وانتظر اكتمال البناء.

### ج. ملاحظات تكامل مهمة
- **الإشعارات (Push):** الواجهة جاهزة فقط؛ التذكيرات الفعلية تحتاج ربطاً بنظام إشعارات خلفي على Manus.
- **تقرير AI الأسبوعي:** إن أُضيف لاحقاً، يجب أن يُنادى عبر Backend آمن (لا استدعاء مباشر من الواجهة).
- **التخزين الحالي:** البيانات في `localStorage` بالمتصفح. لمزامنة عبر الأجهزة، تُربط `useCore` لاحقاً بقاعدة بيانات Manus (نقطة تكامل واحدة).

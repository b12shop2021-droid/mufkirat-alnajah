# تسليم: صفحة "صمّم جدول تمارينك"

تحويل النموذج الأولي `mufkirat-najah-custom-workout.html` إلى React/TypeScript.

## الملفات المعدّلة/المضافة
- `src/core/useCore.tsx` — أنواع `CustomDay`/`CustomExercise` + 7 دوال.
- `src/pages/CustomWorkout.tsx` — الصفحة (إضافة).
- `src/styles/global.css` — أصناف الصفحة (كل لون من `:root`).
- `src/App.tsx` — مسار `/custom-workout`.

## الثغرات/الأخطاء المُصلحة من النموذج الأولي
- 🔴 **XSS** — استبدال `innerHTML` (الذي يحقن أسماء المستخدم) بـ JSX و`{متغير}`.
- 🔴 معرّفات تسلسلية → `crypto.randomUUID()`.
- 🔴 `confirm()` → `ConfirmDialog`.
- 🟡 ألوان ثابتة و`:root` مكرر → `var(--)` من `global.css`.
- 🟡 لا تنظيف مدخلات → `clean()` + `clampNum` (الجولات 0–100، لا سالب).
- 🟡 حالة محلية تُفقد → تخزين في `useCore` (حفظ تلقائي).
- 🟡 رفع صورة بلا تحقق → التحقق من نوع الملف (صورة) + حد حجم 1.5MB.

## ما حُفظ من السلوك الأصلي
- أيام مفتوحة العدد، اسم حر لكل يوم، تمارين حرة (اسم/جولات/تكرارات نصية/صعوبة Select/ملاحظات)، صورة اختيارية لكل يوم، طيّ/فتح، زر «تـم — حفظ» مع احتفال، رسائل تحقق.

## قرار تقني
- التكرارات (`reps`) **نص** (مثل "8-12") عبر نوع `CustomExercise` مستقل — مع إبقاء `Exercise { reps: number }` لصفحة الكابتن سعود لاحقاً (حسب موافقتك).

## قائمة الفحص
- XSS ✅ · ألوان `var()` ✅ · لا تكرار ✅ (`updateCustom` موحّد) · لا `prompt`/`confirm` ✅
- `randomUUID` ✅ · تنظيف مدخلات ✅ · `useCore` مركزي ✅ · تجاوب + احتفال ✅ · البناء ✅
